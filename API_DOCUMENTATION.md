# API Documentation

## Base URL
http://localhost:8000/api

## Endpoints

### GET /tasks
Get all tasks.

### POST /tasks  
Create new task.
Body: {"title": "...", "description": "...", "status": "pending"}

### GET /tasks/{id}
Get single task.

### PUT /tasks/{id}
Update task.

### DELETE /tasks/{id}
Delete task.
